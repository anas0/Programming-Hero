DOM meaning Document Object Model. Which is a programming interface for web documents. In a webpage dom connect document, element, node, style, javascript and make a dynamic webpage.

The purpose of the dom is to connect document to javascript which make webpage to more user interactive which will halp user to comfortable to use.

There are six ways to select elements in a dom.

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. getElementsByName
  5. querySelector
  6. querySelectorAll

Event bubble is a kind of a dom tree and it is a way events handle in the browser. When a element receive a event it go to its parent and goes up to the tree to its root element.
