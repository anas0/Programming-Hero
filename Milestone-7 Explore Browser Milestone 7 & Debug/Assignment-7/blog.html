map executes the same code (Like multiply by a 2) on every element in an array and returns a new array with the updated elements.

filter checks every element in an array to see if it meets the condition and returns a new array with the elements that match the condition.

forEach is used to execute the same code on every element in an array but does not change the array and it dose not return anything.

find checks every element in an array to see if it meets the condition and returns the first element that match the condition.

Local storage allows for larger amounts of data to be stored, but the data is not deleted when the browser is closed. Local storage is useful for storing data that the user will need to access later

Session storage is only stored data for the current session. This means that the data will be deleted when the user closes the browser.

Cookie is a small piece of data that is stored on the user's computer. Cookies are used to store information about the user such as their name, password, and preferences. However, they are limited to 4kb of data
